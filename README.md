# Tamarin Protocols

This repository is for testing and running the protocols defined in our base repository that contains the modified ProVerif engine: [Hafeez-hm/Hash-Gone-Right](https://github.com/Hafeez-hm/Hash-Gone-Right).

To validate correctness, we translate the existing ProVerif .pvl files and verify that Tamarin results back our claims.

- Protocols live in /protocols
- Setup steps are in Setup.md
